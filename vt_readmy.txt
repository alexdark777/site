---------
УСТАНОВКА
---------


Движок не предназначен для конечного пользователя.
Для установки рекомендуется обратиться к спецалисту в области программирования.
Особых знаний не требуется, достаточно выполнить пошаговую установку:

1. Распакуйте архив в корневую папку сайта.
2. Отредактируйте файл db.php в папке inc.
3. Отредактируйте при необходимости файл config.php в папке inc.
4. Импортируйте в базу данных файл inc/INSTALL.sql


-------------------
УСТАНОВКА ЗАВЕРШЕНА
-------------------


Для входа в панель администратора используйте:
1. Страница входа - http://ваш_домен.ru/admin/
2. Логин - administrator
3. Пароль - 12345678

Не пытайтесь использовать данные администратора для входа как обычный пользователь с главной страницы.
У администратора отдельная страница входа - http://ваш_домен.ru/admin/

Для корректной работы движка установите права 777 на папку compiles.
Если при первом входе на сайт сервер выдает ошибку 500, откройте в текстовом редакторе файл .htaccess и удалите строку "Options Multiviews"

Если города не установились в таблицу базы данных SITY автоматически, то импортируйте в базу содержимое файла vt_sitys.sql.
